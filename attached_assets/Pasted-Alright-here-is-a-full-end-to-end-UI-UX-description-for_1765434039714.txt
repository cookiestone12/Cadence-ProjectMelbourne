⭐️ AMPERSOUND — FULL UI/UX DESIGN SPEC (LIGHT VERSION)

Purpose: Refresh the entire app to a premium, modern, industry-level UI

⸻

1️⃣ FOUNDATIONS

1. Color System (Light Mode)

Apple’s light interfaces rely on warm neutrals and soft depth.
Use this exact palette:

Backgrounds
	•	Primary App Background: #F7F7F9 (Apple off-white)
	•	Content Panels / Cards: #FFFFFF
	•	Subtle Contrast Panels: #F2F2F5 (for table rows, sections)

Accents

Use your brand’s purple/pink gradient subtly, Apple-style:
	•	Primary Gradient: #A020F0 → #E540AC
	•	Secondary Purple: #7D7AFF
	•	Highlight Pink: #FF2F71

Functional Colors
	•	Success: #34C759
	•	Warning: #FF9500
	•	Error: #FF3B30
	•	Info: #007AFF (iOS blue)

Shadows
	•	Very soft, almost invisible:
0px 4px 12px rgba(0,0,0,0.08) for cards
0px 1px 3px rgba(0,0,0,0.06) for buttons

⸻

2️⃣ TYPOGRAPHY (Apple System Type)

Use SF Pro only.

Title 1 — 34px, semibold

Used for page headers (Roster, Dashboard)

Title 2 — 28px, semibold

Used for section headers (Creator Profiles)

Title 3 — 22px, medium

Used for card headers (Performance, Financials)

Body — 17px regular

Used everywhere text is meant to be read.

Caption — 13px regular

Used for labels, stat descriptions, table headers.

Spacing rule:
Every block of content breathes — use 24–32px vertical space.

⸻

3️⃣ LAYOUT SYSTEM

Global Layout
	•	Full-width background remains pure white/very light grey.
	•	Content lives inside cards with rounded corners & soft shadows.
	•	Spacing:
	•	Horizontal margin: 20–24px mobile, 40–60px desktop
	•	Vertical rhythm: consistent 24–32px

Card Style
	•	Background: #FFFFFF
	•	Radius: 18px
	•	Shadow: subtle
	•	Padding inside card: 20–28px

Tables (Songs, Placements) appear inside elevated cards, not full width.

⸻
